Pathfinding

Developed an implementation of a Multi-agent pathfinding system in a dynamic environment (C++) and a 3D viewer/editor to visualize the different steps of the algorithm and interact with the agents (Java, Swing).

Use of A* algorithm with a cost based on the manathan distance:


We also implemented a field of view so that the vision of an agent can be restricted to a given distance. In this more realistic situation, an agent only knows obstacles from what it has already visited and what is present in its field of view.

Chemistry game

Used technologies: C#, WPF, WPF 3D.
